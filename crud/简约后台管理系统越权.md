# Vulnerability Introduction：

In Crud Simple Back-end Management System v1.0.0, there is a vulnerability that allows unauthorized modification of administrator personal information in the back-end. Attackers can maliciously modify the personal information of administrators by changing the user ID.

# Vulnerability reproduction：

After logging into the backend and clicking on user management, the generated traffic data packets can reveal the administrator's ID and information of other users.
